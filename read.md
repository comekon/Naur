Naur — Design Brief Lengkap
🧠 Konsep Utama
Naur adalah specialty tea bar premium Indonesia yang menarget anak muda urban usia 18–30 tahun. Nama "Naur" dari bahasa Elvish berarti api — filosofinya: tanpa api, tak ada teh. Tanpa teh, tak ada momen.
Tagline: "Nyalakan Momenmu"
Sub-tagline: "Specialty tea bar untuk jiwa yang berani menikmati."
Diferensiasi: Naur bukan boba shop biasa. Naur adalah tea bar — fokus pada teh daun asli berkualitas tinggi, tanpa topping berlebihan, dengan visual brand yang bold dan berani di tengah kompetitor yang semuanya pakai estetik oriental/minimalis.

🎨 Design System
Color Palette
Primary      : #E63000  (Merah api — bold, berani, energik)
Secondary    : #FF9500  (Oranye bara — hangat, nyala)
Tertiary     : #1A1A2E  (Navy hitam — kontras, premium)
Accent       : #FFE500  (Kuning terang — pop, eye-catching)
Light Base   : #FAFAFA  (Off-white — ruang napas)
Text Dark    : #1A1A1A
Text Light   : #FAFAFA
Text Muted   : #757575

Palet ini terinspirasi dari api dan bara — merah, oranye, kuning. Kontras dengan navy gelap sebagai grounding. Jauh dari estetik teh yang biasanya hijau/coklat/earthy.

Typography
Display/Hero  : "Syne ExtraBold" — geometris, tegas, karakter kuat
Heading       : "Syne Bold"
Body          : "DM Sans Regular"
Label/Tag     : "Space Mono" — monospace untuk harga, kode, badge
Visual Style

Flat design dengan bold outlines — tegas, bukan lembut
Background: putih atau navy gelap sebagai kanvas
Elemen dekoratif: lidah api bergaya geometris, percikan air abstrak
Foto produk: high contrast, vivid — bukan foto teh yang soft dan dreamy
Layout: asymmetric grid — dinamis, tidak monoton
Tombol: solid fill, border-radius 4px — tegas
Shadow: flat offset shadow 4px 4px 0px #1A1A1A — khas bold colorful design
Texture: halftone dots dan grain ringan di hero section


📐 Layout System
Max width container : 1280px
Grid                : 12 kolom
Gutter              : 32px
Border radius card  : 4px
Border radius button: 4px
Shadow card         : 4px 4px 0px #1A1A1A
